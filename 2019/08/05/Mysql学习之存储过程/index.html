存储过程小结

1：基本语法
2：参数 in|out|inout

2.1:IN参数的值必须在调用存储过程时指定，在存储过程中修改该参数的值不能被返回，为默认值

2.2:OUT:该值可在存储过程内部被改变，并可返回

2.3:INOUT:调用时指定，并且可被改变和返回

3.变量赋值

语法：SET 变量名 = 变量值 [,变量名= 变量值 …]

用户变量

用户变量一般以@开头

注意：滥用用户变量会导致程序难以理解及管理

  • 在MySQL客户端使用用户变量
  • 在存储过程中使用用户变量
  • 在存储过程间传递全局范围的用户变量

MySQL存储过程可使用两种风格的注释：

双杠：–，该风格一般用于单行注释

C风格： 一般用于多行注释
